Hierarquia de título

A hierarquia de títulos é a organização das informações de acordo com o conteúdo, usando conscientemente o <h1>, <h2>, <h3> e etc. Que são tags atribuidas para títulos, de nível 1, 2, 3 e etc.

Essas tags não representam que um vai ser maior que o outro e sim o peso do título, como ele vai estar disposto na informação, exemplo disso temos em uma manchete por exemplo.

Percebe-se na imagem abaixo que, existe um título, ou seja, <h1> e também há sub-títulos, que seria um <h2>. Então da a entender que, essas tags não servem para deixar letras maiores ou menores e sim para trazer sentido ao que se está mostrando, onde os diferentes tipos de títulos vão estar dispostos de forma consciente e correta, dependendo da forma de criação do conteúdo.

Os parágrafos também fazem parte dessa hierarquia, por mais que seja de título, pois eles também possuem um lógica e coesão ao serem inseridos. Os parágrafos são usados para expor o conteúdo em si, como o exemplo dessa própria explicação que é um parágrafo. Só pra relembrar, a tag do parágrafo é essa: <p> conteúdo </p>.

Negrito e itálico

Existem duas formas de se colocar uma palavra em negrito e também, duas para se colocar em itálico. A diferença dessas duas formas é que, uma apresenta uma forma não semântica (passe o mouse em semântica para saber o significado) de se atribuir isso, e a outra oferece uma forma semântica.

Negrito

Primeiramente para o negrito. A primeira forma é utilizando a tag <b> palavra </b> e a segunda forma é utilizando a tag <strong> palavra </strong>. As duas tags apresentam a mesma função porém, como o HTML é uma linguagem que utiliza a semântica, ou seja, o sentido correto das palavras, a forma mais correta de se usar é o strong pois ele atribui um significado de que aquela palavra vai ser mais forte (essa é a tradução dele). Já o b ele coloca a palavra em negrito, porém não atribui um significado sobre aquilo, pois n possui uma semântica.

Itálico

Assim como o negrito, também existem duas formas de se colocar uma palavra ou frase em destaque com o itálico, uma apresenta sentido semântico e a outra não. A primeira forma, que é a semântica é usar as tags <em> palavra </em> que deixa a palavra em destaque e passa a função de ênfase na palavra. A segunda forma é utilizando a tag <i> palavra </i> que também me passa o sentido da ênfase, mas não me atribui uma semântica na tag.

OBSERVAÇÃO:
Você pode utilizar as duas formas, tanto no negrito quanto no itálico, mas é importante que saiba essa diferença entre as formas.
