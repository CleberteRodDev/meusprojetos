Primeiros códigos/tags em HTML

A area <head> é a área das configurações da página, chamada área de cabeça.

A área onde vão estar dispostos os códigos de "forma".

A área <body> é a área onde se trabalha o conteúdo em si.

A área onde vai estar disposto os códigos HTML as tags e etc.

Algumas tags HTML:

<h1>título</h1> - Essa tag é usada para se fazer um título de nível um (que será abordado mais a frente).

<h2>título2</h2> - Essa tag é usada para criar um título de nível dois.

<h3>título3</h3> - Essa tag é usada para criar um título de nível 3

<hr> - Serve para criar uma linha horizontal exemplo abaixo(essa tag não possui fechamento).

<!-- comentário --> - Essa tag é usada para fazer um comentário dentro do código, para algum outro DEV que venha a ver seus códigos.

<p>Este é meu parágrafo</p> - Essa tag serve para se criar um parágrafo (conteúdo em parágrafo).

Essas tags são algumas das mais usadas em questão de conteúdo/escrita.

Parágrafo e quebra de linha:

Como vimos acima a tag <p> é usada para se criar um parágrafo. O parágrafo está disposto de várias informações, e muita das vezes ele fica muito grande e acaba tomando parte de toda a tela. Para essa determinada situações a gente tem uma tag usada para a quebra de linha em determinada parte do parágrafo de escolha do escritor. Essa é a tag <br> que é usada para
quebrar uma linha, como acabei de fazer. Geralmente ela é implementada no meio do texto do parágrafo.

Adicionando emojis

É bem comum hoje em dia vemos a presença de alguns emojis em textos, e também podemos adicioná-los com HTML em seu site, utilizando
o &#x seguindo do código do emoji que voce pode pegar no site da emojipedia.

Ao achar o emoji do seu gosto, você deve copiar o código dele ex 1F600, que seria o código de um emoji sorridente. Juntando o &#x com esse código aparece o emoji que você irá utilizar ex: 😀. O mesmo processo é feito caso queira acoplar outros emojis ao seu site, muito simple não é 😍.
