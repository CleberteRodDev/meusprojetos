Listas UL e OL

Listas Ordenadas OL

São listas que apresentam uma sequência lógica e são usadas abrindo as tags <ol> e </ol> e dentro dessas tags vai ter outra tag, onde vão ficar os elementos, que é a tag <li> conteúdo </li> ai dentro dessas tags <li>, vão estar os componentes da lista enumerados, por número, número romano e letra (maiúscula ou minúscula) Essa lista fica disposta dentro de um parágrafo, não se esqueça.

Exemplo de lista ordenada:

  1. Abrir um parágrafo
  2. Usar a tag <ol>
  3. Usar a tag <li>
  4. Escrever o item da lista no li

Abaixo está um modelo de como fica o código:

                    <p>
                        <ol>
                            <li>Abrir um parágrafo</li>
                            <li>Usar a tag ol </li>
                            <li>Usar a tag li <li>
                            <li>Escrever o item da lista no li</li>
                        </ol>
                    </p>

Acima você vê como ficou o código.

Percebe que a lista ordenada segue um padrão lógico. Primeiro isso, depois aquilo e etc, e também da pra ver que possuem numerações, que podem ser trocadas usando um type="A", type="a", type="1", type="I" ou type="i" para letras maiúsculas ou minúsculas na lista se especifica no type, para números romanos maiores ou menores também usa o type e para número usa um type com o número 1. Esses types vão dentro da tag <ol>.
